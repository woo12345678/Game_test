# Top-View Shooter (Unity)
##구현 완료 
1. 3D 기반 탑뷰 시점
2. 플레이어/몬스터: 캡슐 오브젝트
3. 몬스터 2종
   - 근거리 몬스터: 플레이어 추적, 충돌 시 피격 판정
	{계속 붙어있어도 1.4초에 한번 데미지가 들어가도록 추가하였습니다}
   - 원거리 몬스터: 주기적으로 투사체 발사
4. 플레이어
   - WASD 이동
   - 마우스 클릭 시 가장 가까운 몬스터에게 구형 투사체 발사
     {플레이어 스크립트에있는 isAuto 가 트루라면 자동으로 몬스터 조준해서 투사체 발사
      펄스라면 마우스클릭한 방향으로 발사 inspector 에서 isAuto 조절가능 }
   - 플레이어 상단 체력바 표시
5. 카메라: 플레이어 트래킹
6. 필드: 경계 처리로 이탈 불가
 구현완료 


## Unity Version
- Unity 6000.0.58f2 

## How to Run
 SampleScene 실행
빨간색 : 근거리 적군 X 2
보라색 : 원거리 적군 X 1
